## 使用说明

请根据具体脚本的功能和要求来使用。大部分Python脚本可以通过以下方式运行：

```bash
python3 storage/script_name.py
```

Shell脚本可以通过以下方式运行：

```bash
chmod +x storage/script_name.sh
./storage/script_name.sh
```

- 部分脚本可能需要特定的环境变量或配置文件
